Mechanism

How the Renderer Works

Scene Graph

A THREE.Group holds all CelestialBody instances. Each body owns an orbitGroup that positions it relative to the parent star.

Animation Loop

updatePlanet(delta) is called each frame. It advances every body's orbital angle and applies per-body rotation speed, producing continuous motion.

Instanced Mesh

The 5,000-body asteroid belt uses THREE.InstancedMesh for a single GPU draw call — keeping frame-rate stable regardless of body count.

Emissive Glow

The Sun uses an emissive MeshBasicMaterial overlaid with a semi-transparent SphereGeometry at 4.7 units radius to simulate corona glow.
